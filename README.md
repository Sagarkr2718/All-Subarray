# All-Subarray
Given an integer array A of length N. Print all possible subarray.
Problem Constraints

1 <= N <= 100
1 <= A[i] <= 103
Input Format

First line is N, size of array
Next line contains N integers
Output Format

In each line print a subarray with its elements separated by space
Example Input

Input 1:
3
6 7 1
Input 2:
4
18 14 10 14
Example Output

Output 1:
6 
6 7 
6 7 1 
7 
7 1 
1 
Output 2:
18 
18 14 
18 14 10 
18 14 10 14 
14 
14 10 
14 10 14 
10 
10 14 
14
Example Explanation

Explanation:
Each line contain a possible subarray
